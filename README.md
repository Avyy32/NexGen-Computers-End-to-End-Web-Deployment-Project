# NexGen Computers – Business Website

A professional multi-page business website built for a local computer services store.
This project covers the complete lifecycle of a real-world website — from design and development to live deployment with a custom domain and SSL.

## Project Overview
NexGen Computers required a clean, modern, and user-friendly website to showcase their services, products, and contact information.  
The focus was on clarity, responsiveness, and real production readiness rather than just static design.

## Features
- Responsive multi-page layout (Home, About, Services, Products, Reviews, Contact)
- Clean and modern UI suitable for a computer services business
- Service and product showcase sections
- WhatsApp-based enquiry flow
- Mobile-friendly and optimized layout

## Tech Stack
- HTML
- CSS

## Deployment & Hosting
- Hosted on **Netlify**
- Custom domain integration
- DNS configuration handled via GoDaddy
- SSL (HTTPS) enabled using Let’s Encrypt
- Production-ready deployment

## Live Website
https://nexgencomputer.in

## Learning Outcomes
This project provided hands-on experience with:
- End-to-end website deployment
- DNS record configuration and propagation
- SSL verification and HTTPS setup
- Real-world troubleshooting during production deployment
